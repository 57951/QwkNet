# Support

## Before you ask
- Check `/docs/building.md` for build issues.
- Check the wiki for common problems.
- If a packet fails to parse, run the diagnostics tool and include its output.

## What to include in an issue
- OS and .NET version
- Steps to reproduce
- Sample packet (if you can share it) or a minimal reproduction
- Diagnostics output (redact personal data)

## Out of scope
- Compiling and using on Linux, Android, or iOS (it may work, but you're on your own!)
