# Contributing

Thanks for considering contributing to QWK.NET.

## Ground rules
- Keep the core library dependency-free.
- Preserve on-disk bytes wherever possible.
- Follow repository coding standards and documentation style rules.
- Use British English in documentation and XML comments.
- Indents are 2-spaces, not four, not tabs.
- Avoid discussions related to politics, religion, and social issues.

## Archive Extensions
- Archive extensions are not to be contributed. You can open an issue to share a link to your extension, but it will not be included in the library or this repository.

## Pull request expectations
- Keep changes scoped.
- Update documentation when behaviour changes.
- Add or update tests where relevant.
